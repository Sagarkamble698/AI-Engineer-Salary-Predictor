# 🚀 AI Engineer Salary Predictor

An advanced beginner-friendly Machine Learning web application built using Gradio and scikit-learn.

This project predicts the estimated salary of an AI Engineer based on:

* Years of Experience
* Technical Skills
* Number of Projects

The application provides a clean interactive UI and real-time salary prediction using Machine Learning.

# 💡 How It Works

1. User enters:

   * Experience
   * Skills Score
   * Projects Completed

2. The Machine Learning model processes the input.

3. The app predicts estimated salary in LPA.
